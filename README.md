# Express API Backend Project

In the project directory, you can run:

### `git clone https://github.com/Jigyasa08/Teric_Task_Backend.git`

### `cd Teric_Task_Backend/`

### `npm install`

### `npm run start`

## Back End Technology

* Express

## Database Development Technologies
